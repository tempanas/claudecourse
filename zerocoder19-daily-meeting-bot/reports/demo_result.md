# Ожидаемый результат демо

## Команда `/today`

Бот читает обезличенные данные из `data/sample_events.json` и показывает три
встречи на текущий день:

1. Консультация тарифа Бизнес для `student_001` - Zoom найден.
2. Q&A по потоку `stream_demo` - Zoom найден.
3. Проверка расписания встречи - Zoom не найден, событие требует проверки.

Для каждой встречи выводятся название, время, участники и статус Zoom-ссылки.

## Команда `/tomorrow`

Бот показывает две встречи:

1. Консультация с `curator_a` - Zoom не найден, событие требует проверки.
2. Внутренняя встреча - Zoom найден.

## Польза для Зерокодер19

Демо показывает основной рабочий сценарий без доступа к реальному календарю и
без секретных ключей. Зерокодер19 быстро видит расписание, отличает готовые к
проведению встречи от событий без Zoom и понимает, какие записи нужно открыть
и проверить в календаре.
