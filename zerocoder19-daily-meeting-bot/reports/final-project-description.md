# Описание финального проекта

## Название

Zerocoder19 Daily Meeting Bot

## Проблема

Зерокодер19 контролирует живые встречи, консультации, Q&A и Zoom-созвоны. В
течение дня нужно быстро понимать, какие события стоят в календаре и есть ли в
них ссылка для подключения. Если Zoom-ссылка не добавлена, встреча может
сорваться или потребовать срочной ручной проверки.

## Пользователь

Пользователь проекта - специалист в должности Зерокодер19. Ему нужно видеть
расписание встреч на сегодня и завтра, а также быстро находить события, где
нет Zoom-ссылки.

## Основная функция

Telegram-бот показывает встречи на сегодня или завтра и для каждой встречи
проверяет наличие Zoom-ссылки.

Если ссылка найдена, бот выводит статус `Zoom: найден ✅`. Если ссылка не
найдена, бот выводит `Zoom: не найден ⚠️` и действие: проверить событие в
календаре.

## Формат результата

Результат выводится сообщением в Telegram:

- дата;
- количество встреч;
- название события;
- время начала и окончания;
- участники, если они есть;
- статус Zoom-ссылки;
- действие для событий без Zoom.

## Минимальный функционал

- команда `/start`;
- команда `/help`;
- команда `/health`;
- команда `/demo`;
- команда `/today`;
- команда `/tomorrow`;
- демо-режим на `data/sample_events.json`;
- Google Calendar режим через OAuth Desktop app;
- поиск Zoom-ссылок в описании, месте, `hangoutLink` и
  `conferenceData.entryPoints`;
- безопасное хранение локальных секретов вне Git.

## Технологии

- Python 3.9+;
- python-telegram-bot;
- Google Calendar API;
- google-auth-oauthlib;
- python-dotenv;
- JSON;
- Markdown.
