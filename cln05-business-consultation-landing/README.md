# Запись на консультацию тарифа Бизнес

Локальный демонстрационный лендинг для домашнего задания CLs02, урок 5 «Разработка конверсионного сайта».

## Как открыть сайт

Самый простой способ:

1. Откройте папку проекта.
2. Дважды нажмите на файл `index.html`.
3. Страница откроется в браузере и будет работать локально.

Также можно запустить простой локальный сервер из папки проекта:

```bash
python3 -m http.server 8000
```

После запуска откройте в браузере адрес:

```text
http://localhost:8000
```

Для остановки сервера нажмите `Ctrl+C` в терминале.

## Файлы проекта

- `index.html` — структура и тексты лендинга.
- `styles.css` — оформление и адаптивная вёрстка.
- `script.js` — локальная валидация демонстрационной формы.
- `SPEC.md` — спецификация проекта.
- `Организация_консультации_Зерокодер19.pdf` — исходный рабочий регламент.

## Безопасность

- Форма отменяет стандартную отправку и не выполняет сетевых запросов.
- Введённые данные не сохраняются в браузере, файлах или внешних сервисах.
- На странице нет реальных контактов, рабочих ссылок и платёжных данных.
- Проект предназначен только для локального запуска и не требует деплоя.
